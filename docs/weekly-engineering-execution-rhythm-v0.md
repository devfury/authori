# Weekly Engineering Execution Rhythm v0

목표: 매주 계획-실행-회고 루프를 고정해 진행률/리스크/다음 액션을 투명하게 유지한다.

## 1) 주간 계획 코멘트 포맷 (월요일)
- 주간 목표 1~3개
- 이번 주 완료 기준(DoD)
- 예상 리스크/의존성
- 담당자 및 우선순위

## 2) 주중 진행률 업데이트 규칙 (수/목)
- 완료/진행/지연 항목 구분
- 리스크 발생 시 즉시 blocker와 요청 주체 명시
- 새로운 작업이 생기면 관련 이슈 링크 추가

## 3) 주말 회고/차주 이관 규칙 (금요일)
- 이번 주 달성률(완료/미완료)
- 미완료 원인 1~2개
- 차주 이관 항목 및 선행조건
- 의사결정 필요 항목(CEO/QA 등)

## 4) 1회차 실행 예시
- 계획: 온보딩 체크리스트 v0, PR/DoD 템플릿 v0, 주간 리듬 문서화
- 진행: [EZC-30](/EZC/issues/EZC-30) 완료, [EZC-31](/EZC/issues/EZC-31) 진행
- 리스크: 리뷰 기준 합의 지연 가능
- 다음 액션: [EZC-32](/EZC/issues/EZC-32) 코멘트로 루프 고정
